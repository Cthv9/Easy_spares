# Configuratore Kit Ricambi Generatori Diesel

Configuratore statico per individuare il kit ricambi corretto in base al modello del generatore diesel e visualizzare la distinta articoli del kit.

---

## Funzionalità

- Selezione guidata del modello generatore diesel
- Ricerca rapida del modello
- Identificazione automatica del kit corretto
- Visualizzazione tabellare degli articoli del kit
- Copia della tabella formattata negli appunti, utile per incollarla direttamente nel corpo di una mail
- Supporto PWA installabile
- Funzionamento offline dopo il primo caricamento
- Deploy statico, senza backend
